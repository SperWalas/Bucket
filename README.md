
Bucket
======================

Projet Startup weeks


## Installation

Installer les packages node et bower.

`npm install`

`bower install`



##Dependencies :

- RequireJS
- BackboneJS
- jQuery
- Lodash
- Grunt
- almond
- GSAP


##APIs




## LiveReload

Utiliser grunt pour compiler le scss.

`grunt watch`



## Start Server & Livereload

Lance le serveur sur le localhost:9000 & Livereload

`grunt server`



## Task clear

`grunt clean`



## Task SASS

Compile sass en css

`grunt sass`
